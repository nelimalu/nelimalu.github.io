[comp sci] / recursion

Recursion is a process in which a method calls itself repeatedly until a particular condition is met. In Java, recursion is a useful technique that can be used to solve various problems.

Example 1: Factorial of a Number

The factorial of a number is the product of all positive integers less than or equal to that number. The factorial of 0 is 1. The factorial of a number n can be calculated using the following formula: 
n! = n*(n-1)*(n-2)*...*3*2*1
The following code shows how to calculate the factorial of a number using recursion in Java: 
public static int factorial(int n) {
    if (n == 0) {
        return 1;
    } else {
        return n * factorial(n-1);
    }
}
In the above code, the factorial() method calls itself with n-1 until n becomes 0. When n becomes 0, the recursion stops and the final result is returned.

Example 2: Fibonacci

The Fibonacci series is a series of numbers in which each number is the sum of the two preceding numbers. The first two numbers in the series are 0 and 1. The Fibonacci series can be calculated using the following formula: 
Fib(n) = Fib(n-1) + Fib(n-2)
The following code shows how to calculate the Fibonacci series using recursion in Java: 
public static int fibonacci(int n) {
    if (n <= 1) {
        return n;
    } else {
        return fibonacci(n-1) + fibonacci(n-2);
    }
}
In the above code, the fibonacci() method calls itself with n-1 and n-2 until n becomes 1 or 0. When n becomes 1 or 0, the recursion stops and the final result is returned. Recursion can be a powerful tool in solving complex problems in Java. However, it is important to use it carefully to avoid infinite loops and stack overflows.
